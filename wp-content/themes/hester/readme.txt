== Description ==
Hester is lightweight, modern, highly customizable multi-purpose translatable theme. Now it is easy for anyone want to create their awesome website. It's perfect for any type of website blogs, businesses, online shops, creatives, hotels, restaurants, startups and more.With this theme you get perfect blend of performance, modern design powerful yet simple to use options.It comes with unlimited color schemes, pre-built demos, woo-commerce integration, multiple layouts and many more. It works perfectly with new Gutenberg editor and other page builders.Theme is a Cross-Browser Compatible thats works on All leading web browsers as well as smart phones.
